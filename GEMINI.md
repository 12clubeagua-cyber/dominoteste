# Instrucoes de Operacao do Projeto (GEMINI.md)

- **Correcoes de Codigo:** Apenas aplique alteracoes que resolvam bugs comprovados. Evite refatoracoes, alteracoes de estilo ou "limpezas" se o codigo atual estiver funcionando corretamente.
- **Padrão de Texto:** Mantenha todo o texto, comentarios e strings internas em padrao ASCII (sem acentos ou caracteres especiais) para garantir compatibilidade total.
- **Verificacao:** Antes de qualquer alteracao, garanta que a funcionalidade existente nao seja afetada.
- **Documentacao:** Siga este guia em todas as interacoes de desenvolvimento para garantir a estabilidade do sistema.
