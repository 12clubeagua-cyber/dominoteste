# Economia e Simulação - X4: Foundations

## O Mercado Real
Diferente de outros jogos, no X4 cada nave e estação é construída a partir de recursos físicos. Se um transporte de 'Hull Parts' é destruído, o estaleiro pode ficar sem material para construir novas naves.

## Cadeia de Produção
1. **Mineração**: Extração de Minério, Silício, Hidrogênio.
2. **Refino**: Transformação em Grapheno, Metais Refinados, etc.
3. **Componentes**: Produção de Peças de Motor, Microchips, Partes de Casco.
4. **Produto Final**: Construção de Naves e Estações.

## Lucro e Comércio
- **Trade Computer Extension**: Essencial para ver preços remotamente.
- **Satélites**: Coloque-os perto de estações para manter os dados de preços atualizados.
- **Autotrade**: Comando avançado para naves com capitães experientes.
